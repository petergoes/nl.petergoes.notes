The `.class` selector matches any `<element>` with the specified `class` attribute

```css
.brown {
	background-color: #DBA67B;
	padding: 1rem;
}

.red {
	background-color: #A55C55;
	padding: 1rem;
}
```

```html
<div class="brown">
	Element with class brown
</div>

<div class="red">
	Element with class red
</div>
```
