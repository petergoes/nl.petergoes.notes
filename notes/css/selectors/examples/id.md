The id selector matches the `<element>` with the specified `id`

```css
#my-id {
	background-color: #236475;
	padding: 1rem;
}

#my-other-id {
	background-color: #001D29;
	padding: 1rem;
}
```

```html
<div id="my-id">
	Element with an id of my-id
</div>

<div id="my-other-id">
	Element with an id of my-other-id
</div>
```
